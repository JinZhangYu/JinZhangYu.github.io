Posted on 2018-09-18

  1. Mac Port的下载地址
    http://www.macports.org/install.php

  2. Mac Port的说明文档
    http://guide.macports.org/

  3. Mac Port中第三方软件下载包存放的默认路径是：/opt/local/var/macports/distfiles/ 为了提高安装速度，可以在安装新port时直接将此目录下的文件拷贝到新的Mac Port相同的目录中就可以避免Port去网上下载。

port常用命令列表

  • help

    • help通常用来查看某个指令的帮助说明

  • selfupdate

    • 同步本地和全球的软件树,有必要时,同时升级mac port自己.

  • sync

    • 同步本地和全球的ports tree,但不检查自己是否有更新.

  • list

    • 列出当前所有的可用软件,如果想查找是否有自己想要的软件时,还是使用search指令方便一些.

  • search

    • 模糊搜索,可以匹配软件名字和描述,还有更高级的用法,具体看port help search

  • info

    • 查看一款软件的详细信息

  • deps

    • 查看一款软件的依赖关系

  • variants

    • 在安装软件前,用这个命令查看软件是否有多个版本.再选择安装一个合适的版本.

  • install

    • 安装软件命令,安装前最好使用variants命令查看是否有多个不同版本.

  • clean

    • 删除一些编译软件时留下的临时文件.

  • uninstall

    • 卸载软件命令
    • 如果这个软件依赖与另外的一款软件,默认不删除它依赖的软件, 使用参数 -f (force) 可以强行删除它依赖的软件.

  • contents

    • 显示软件安装后的文件列表.

  • installed

    • 列出全部或者指定的已经安装的软件.

  • outdated

    • 查看已经安装的软件是否有更新,在执行这个指令前,先执行selfupdate 或者 sync更新软件树

  • upgrade

    • 更新软件,默认一起更新它依赖的所有软件,如果想不更新它依赖的软件,使用 -n 参数

    • 默认不删除旧软件版本,只是使旧软件变成无效状态,如果想要一起删除旧软件,使用 －u 参数

    • 更新所有的可更新软件

    • 更新软件同时删除旧版本软件

  • dependents

    • 查看哪些软件时依赖与这个软件的.删除一个软件时候，最好先执行一下这个命令.
