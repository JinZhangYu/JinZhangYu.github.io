Common Arguments

绘制形状的函数有一些共同的参数，提前在此说明一下：

  • img : The image where you want to draw the shapes
  • color : Color of the shape. for BGR, pass it as a tuple, eg: (255,0,0) for blue. For grayscale, just pass the scalar value.
  • thickness : Thickness of the line or circle etc. If -1 is passed for closed figures like circles, it will fill the shape. default thickness = 1
  • lineType : 线条的类型，具体分为：
    • LINE_8 (or omitted) 8-connected line.
    • LINE_4 - 4-connected line.
    • LINE_AA - antialiased line.
      详细的解释见 八连通、四连通线 以及 抗锯齿线

Drawing Line

对于cv2.line函数，需要有如下注意点，但是一般都不会碰到下述问题：

  • The line is clipped by the image boundaries.
  • For non-antialiased lines with integer coordinates, the 8-connected or 4-connected Bresenham algorithm is used.
  • Thick lines are drawn with rounding endings.
  • Antialiased lines are drawn using Gaussian filtering.

经验之谈：所有绘图函数均会直接影响原图片，这点要注意
